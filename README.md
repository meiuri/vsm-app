## 💡 アプリの概要・作った背景
研究室ではこれまで、先輩から受け継がれたPythonコードでVSM（振動試料型磁力計）のデータ解析を行っていました。しかし、CUIでの操作や、条件を変えるたびに直接コードを書き換える手間があり、日常的なデータ確認において「使いにくさ」という課題がありました。

「実験直後にサクッと生データをプレビューし、複数の試料データを直感的に比較したい」という思いから、従来の解析ロジックを活かしつつ、ブラウザで軽量に動く本アプリを開発しました。

**【開発のこだわり：楽しさとモダンなUI/UX】**
単なる機能のGUI化にとどまらず、**「毎日のルーチンワークが少しでも楽しくなるように」**というコンセプトで設計しています。
UI**: ピンクを基調としたモダンなカスタムCSSを採用。
- **「いいね！💖」機能**: 気に入った補正データを一時ストックし、あとから自由に比較可能。
- **アルバム保存機能**: 比較したグラフの組み合わせをコレクションとして永続保存・復元できるシステムを実装しました。

## ✨ 主な機能とアルゴリズム
- **生データ(.vsm)のダイレクト読み込み**: ヘッダー（上部40行）を自動スキップし、多周期サイクルデータを自動で往路・復路に分割（セグメンテーション）。
- **高度な自動補正アルゴリズム**: 
  - **H軸・M軸の非対称ズレ補正**: 往路・復路のデータの最小二乗誤差を計算し、測定時の磁場・磁化のズレを自動補正。
  - **反磁性成分の除去**: 高磁場領域の傾きから反磁性成分を算出し、バックグラウンドを除去。
  - **反磁界補正**: 試料の形状（面積・膜厚）に基づく反磁界係数（N）を用いた補正。
- **インタラクティブな重ね合わせグラフ**: Plotlyを用いた操作性の高いグラフ描画（表示領域のオンオフ、マウスでの拡大縮小）。
- **ローカルDBによる状態管理**: `pickle` を用いてお気に入りデータやコレクション（アルバム）を永続保存し、次回起動時にも復元可能。
- **ワンクリック一括エクスポート**: 重ね合わせた複数サンプルのデータを、Origin解析用のTSVテキスト、またはCSV形式で即座に出力。

## 🛠 使用技術
- **Python 3.10**
- **Streamlit**: Webアプリケーションフレームワーク（カスタムHTML/CSSによるUI構築）
- **Pandas / NumPy**: データの読み込み・配列計算・自動補正・線形近似処理
- **Plotly**: インタラクティブグラフの描画
- **Pickle**: ローカルDBとしてのデータ永続化
- **PIL / Base64**: カスタムUI用の画像エンコード処理
